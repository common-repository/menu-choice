== Description ==

Easily select a menu to be displayed in a widgetable area. Whether it is a page, post, or custom post type, each page can have a custom menu in the chosen sidebar.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to Appearance > Widgets and drag the 'Menu Choice' widget to the desired widget area.
4. Go to Appearance > Menus and create a menu that you want displayed on a specific page or post.
5. When editing that page, select the menu name from the Menu Choice meta box on the side and update the page.
6. The selected menu will be displayed in the sidebar. Create different menus for different pages.
